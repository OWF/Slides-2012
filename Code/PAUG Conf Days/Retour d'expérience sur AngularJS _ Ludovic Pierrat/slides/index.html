AngularJS, c'est quoi ?

  • un framework Javascript pour construire des web apps
  • un MVC exécuté dans le browser
  • du data-binding automatique entre le modèle et la vue
  • de l'injection de dépencence
  • des tests unitaires et end-to-end

AngularJS, en faits

  • développé chez Google
  • créé en 2009
  • open source (MIT)
  • communauté et équipe de développement actives
  • DoubleClick for Advertisers développé en AngularJS (200 000 lignes de code)

Pourquoi avoir créé AngularJS ?

  • besoin de simplifier la descente d'une partie de la logique applicative sur le client
  • de + en + de possibilités côté client :
    • rapiditité de JS
    • fonctionnalités
    •

    Pourquoi utiliser AngularJS ?

  • une interface utilisateur très rapide
  • une UI très réactive (binding)
  • séparation marquée présentation / données
  • primauté au HTML (vs Javascript) pour les vues
  • moins de code
  • modularité et extensibilité
    •

    Développement Web ?

  • Web 0.0 : HTML + Scripts Serveurs
  • Web 1.0 : HTML + Scripts Serveurs + un peu de Javascript
  • Web 2.0 : HTML + Scripts Serveurs + Ajax + librairies JS
    •

    Développement Web ?

    le changement, c'est maintenant :

    • HTML 5 + MVC pour la navigation et la présentation
    • API REST pour la persistence
    • Scripts serveur pour certaines tâches
